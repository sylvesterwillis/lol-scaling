This is a League of Legends champion info viewer written using React.
This project is currently hosted at [https://well-wave.hyperdev.space/](https://well-wave.hyperdev.space/)

Your Project
------------

On the back-end,
- your app starts at `server.js`
- add frameworks and packages in `package.json`
- safely store app secrets in `.env`

On the front-end,
- edit `client.js`, `style.css` and `index.html`
- drag in `assets`, like images or music, to add them to your project
